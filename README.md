# Debathea

App móvil de **debates en vivo** con sistema de turnos + sección de análisis estilo Reddit.

## Funcionalidades actuales
- Debate en vivo entre 2 participantes
- Dos modos: **Por Mensajes** o **Por Tiempo**
- Chat de espectadores (panel deslizable desde abajo)
- Sección para publicar análisis del debate
- Diseño mobile-first con Tailwind CSS
- Totalmente funcional sin backend (simulado)

## Cómo probarla
1. Abre el archivo `debathea.html` en cualquier navegador (Chrome recomendado).
2. Cambia entre modos (Mensajes / Tiempo).
3. Escribe argumentos en el debate.
4. Publica análisis abajo.
5. Abre el chat de espectadores tocando el botón inferior.
